# temp_manage

一个自用的模版管理工具。

欢迎大家在issue下推荐好用的项目模版，如果真的好用会内置到temp_manage的模版列表中

# 安装

```shell
npm i temp_manage -g
```

```shell
 tm create projectName
```

# Future

- [ ] 支持自定义增加模版 `add <templateName>|<templateAddress>`
- [ ] 支持自定义删除模版 `remove | del <templateName>`
- [ ] 支持设置`模版列表`地址 `tm set source=<url>`
